# Security Policy (Alpha)

## Reporting a Vulnerability

If you discover a security issue, please report it privately.

- Contact: byron@biobitworks.com
- Subject: [Antigence Alpha Security]

Please include:
- Description of the issue
- Steps to reproduce
- Potential impact
- Suggested mitigation (if available)

## Scope

This alpha release contains documentation and manuscript drafts only. It does not ship executable code.

## Response Targets

- Acknowledgment within 7 days
- Status update within 14 days

## Disclosure

We prefer coordinated disclosure. Public disclosure should wait for an agreed fix or mitigation plan.
